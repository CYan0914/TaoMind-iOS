## 🌐 需要托管的页面

必须有两个可公开访问的网页（建议使用 GitHub Pages 免费托管）：

| 页面 | URL | 文件 |
|------|-----|------|
| **Privacy Policy（隐私政策）** | `https://taomind.app/privacy` 或 `https://你的用户名.github.io/taomind/privacy` | `privacy_policy.md` |
| **Terms of Service（服务条款）** | `https://taomind.app/terms` 或 `https://你的用户名.github.io/taomind/terms` | `terms_of_service.md` |

### 托管方式选项：

**方案 A：GitHub Pages（免费，推荐 ⭐）**
1. 创建仓库 `你的用户名/taomind-privacy`
2. 启用 GitHub Pages
3. 将 MD 转为 HTML 或直接上传 PDF
4. 在 App Store Connect 填入 `https://你的用户名.github.io/taomind-privacy/privacy.html`

**方案 B：注册 taomind.app 域名**
1. 在 Namecheap/GoDaddy/阿里云 购买 `taomind.app`（约 ¥30-50/年）
2. 用 GitHub Pages + 自定义域名托管
3. 更新 App 内链接 — 但目前代码已硬编码了 `taomind.app` URL，所以这是最自然的方案

**方案 C：使用隐私政策生成器**
- 如：PrivacyPolicies.com、Iubenda（可托管页面）
- 生成后填入链接
